# AI Lead Scoring Agent

This project implements a reproducible AI-driven lead intelligence pipeline
to identify, enrich, and rank high-intent scientific leads for 3D in-vitro
model adoption.

## Workflow
1. Identify relevant profiles
2. Enrich profiles with publication signals
3. Apply explainable propensity scoring
4. Output ranked leads

## Execution
Run the Google Colab notebook top-to-bottom.

## Output
- Ranked CSV file
- Google Sheet generated from code
